# Pro Edition Features (Inactive in Free v1)

This folder contains Pro Edition features that are **inactive by default** in the Free Edition.

## Status: Inactive Unless Enabled

The Free Edition does not use these modules. They are present for:
- Future Pro Edition compatibility
- Code structure consistency
- Optional organizational policy hooks

## Pro Features (Not Active in Free v1)

- `enforcement.py` - Organization-wide policy enforcement
- `policy_loader.py` - Load and validate org policies
- `policy_schema.py` - Policy schema definitions

## Activation

Pro features require:
1. Valid Pro license
2. Organization policy configuration
3. Explicit enablement in settings

**Free Edition users can safely ignore this folder.**
